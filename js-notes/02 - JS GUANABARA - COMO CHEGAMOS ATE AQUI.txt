JAVA E JAVASCRIPT NÃO TEM NADA A VER.
As duas linguagem são baseadas na linguagem C.

A microsoft vendo a sacada que o JavaScript era, decidiu copiar e fazer o Jscript, implementando algumas melhoras.
Mas não passava de um plagio do projeto.

Porem a NETSCAPE foi mais rapido e conseguiu a padronização da linguagem, uma empresa europeia chamada ECMA, associação europeia de fabricante de computadores.

E com isso, o Javascript cedendo o código para a ecma, surgiu a linguagem ECMASCRIPT.

Basicamente o ecmascript é o javascript padronizado.

Estudar ECMA significa que voce vai estudar a versao padronizada do JS.
